DESCRIPTION  (max 4000 chars)
----------------------------------------
CipherForge turns secret messages into an adventure. Choose from over a dozen cipher modes, chain them together for maximum mystery, or build your very own personal code in The Forge.

🔐 CIPHER MODES INCLUDED:
• Caesar Cipher — classic letter shift
• Vigenère Cipher — keyword-powered encryption
• Atbash — ancient A=Z, B=Y reversal
• Rail Fence — zigzag message scrambling
• Pig Latin — the classic playground code
• Morse Code — dots and dashes like a telegraph operator
• Substitution — swap every letter with a custom alphabet
• Reverse — read it backwards
• Number Cipher — replace letters with numbers
• Elvish Script — mystical runic encoding
• Invisible Ink — hide your message with decoy letters
• Levi's Conundrum — alternating number substitution
• The Caleb Protocol — swap every pair of letters
• The Forge — design your own code where every letter maps to your secret symbol or word

⚒️ THE FORGE
Create a completely personal cipher. Assign any word, symbol, or string to each letter of the alphabet, then share your custom codebook with friends so they can decode your messages.

🔗 CHAIN CIPHERS TOGETHER
Build multi-layer codes by stacking ciphers. Vigenère → Rail Fence → Caesar? Your message becomes nearly impossible to crack without knowing the full chain.

📡 SHARE WITH FRIENDS
Generate a QR code or short text share code for any cipher mode. Friends scan or enter the code to import your exact cipher setup — no accounts needed.

📜 MESSAGE HISTORY
Every message you encode or decode is saved locally so you can refer back to it later. Clear it any time.

🔒 COMPLETELY PRIVATE
CipherForge works 100% offline. No accounts. No ads. No tracking. No data ever leaves your device.

Perfect for kids learning about cryptography, families playing secret message games, teachers introducing code-breaking concepts, and anyone who loves puzzles.
